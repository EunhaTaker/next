# 目标
使用Tauri 2.0开发跨Windows/MacOS的，显眼方式提醒自己当前任务的软件
# 功能
## 任务管理
- [x] 编辑todo list
- [x] 任务可设置截止时间
- [x] 重要程度、紧急程度
- [x] 任务支持选取操作系统的某个桌面进行绑定，且支持切换到对应桌面
## 悬浮窗
- [x] 实现为置顶小窗口，并停靠在显示器右侧中间
- [x] 显示当前专注的任务，简称“专注任务”
- [x] 根据任务数量动态调整高度
- [x] 可拖动改变纵向位置，可移到屏幕左侧
- [x] 底部显示狭长的按钮，按钮中间一个"+"，在中间弹窗显示未添加为专注任务的任务列表以供添加到专注任务，点击添加追加到专注任务列表顶部，也可直接拖到专务任务列表以设置位置
- [x] 每项任务左侧显示上移/下移按钮，可调整专注任务顺序，两个按钮一上一下，共同形成一个正方形
- [x] 上移/下移按钮右侧显示置顶按钮（双重上箭头）
- [x] 任务右侧按钮切换到该任务对应桌面，切换后本悬浮窗也跟随到该桌面
