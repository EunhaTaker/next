# 目标
使用Tauri 2.0开发跨Windows/MacOS的，显眼方式提醒自己当前任务的软件
# 功能
## 任务管理
- [x] 编辑todo list
- [x] 任务可设置截止时间
- [x] 重要程度、紧急程度
- [x] 任务支持选取操作系统的某个桌面进行绑定，且支持切换到对应桌面
- [x] 任务支持创建子任务（子任务创建界面与普通任务相同，不能为子任务创建子任务）
- [x] 子任务支持单独绑定桌面，未绑定的情况下默认使用父任务的桌面
- [x] 支持将子任务单独设为专注任务
- [x] 在主界面可以展开普通任务，查看子任务
- [x] 任务/子任务支持开始/暂停计时：开始计时触发新的计时，暂停计时和完成任务会结束并保存这段计时，支持预览任务使用的所有计时时段
## 悬浮窗
- [x] 实现为置顶小窗口，并停靠在显示器右侧中间
- [x] 显示当前专注的任务，简称"专注任务"
- [x] 根据任务数量动态调整高度
- [x] 可拖动改变纵向位置，可移到屏幕左侧
- [x] 底部显示狭长的按钮，按钮中间一个"+"，在中间弹窗显示未添加为专注任务的任务列表以供添加到专注任务，点击添加追加到专注任务列表顶部，也可直接拖到专务任务列表以设置位置
- [x] 每项任务左侧显示上移/下移按钮，可调整专注任务顺序，两个按钮一上一下，共同形成一个正方形
- [x] 上移/下移按钮右侧显示置顶按钮（双重上箭头）
- [x] 任务右侧按钮切换到该任务对应桌面，切换后本悬浮窗也跟随到该桌面
- [x] 在顶栏增加按钮，支持切换"拆分视图"：只显示第一个任务和它的子任务，默认视图不显示子任务
- [x] "拆分视图"中，有子任务则只聚焦子任务
- [x] "拆分视图"中，子任务也能上移/下移/置顶/计时/切换桌面
- [x] 子任务完成后在悬浮窗不显示；任务完成后从专注任务列表中移除，并从拆分视图切换到默认视图
- [x] 每项任务/子任务增加开始/暂停（计时）按钮，并且鼠标悬停在按钮上可以预览任务使用的所有计时时段
## 快捷键（我提供windows的版本，你帮我自动适配macOS的版本）
- [x] ctrl+alt+h: 隐藏/显示并聚焦到悬浮窗 
- [x] ctrl+alt+{数字}: 显示并聚焦到悬浮窗，并定位到第{数字}个专注任务（从0开始），此后操作系统聚焦在此窗口，且被聚焦的专注任务会高亮或者描边，注：在“拆分视图”中，只聚焦或操作子任务
- 聚焦后进一步快捷键：
    - [x] ctrl+alt+{数字}: 重新定位到某任务
    - [x] UP/DOWN: 聚焦到其他专注任务
    - [x] alt+UP/DOWN: 上移/下移专注任务
    - [x] d: 切换到该任务对应桌面
    - [x] 两次d: 完成任务
    - [x] c: 取消专注
    - [x] esc: 取消选中任务
- [x] t: 切换到拆分视图
- [x] ctrl+alt+space 对第一个任务开始/暂停计时，在"拆分视图"中改为操作第一个子任务
- [x] ctrl+alt+d 对第一个任务/子任务，切换到该任务对应桌面（无绑定桌面则使用父任务的）
- [x] ctrl+alt+m 打开/关闭主界面
## 设置
- 番茄土豆（番茄钟）
    - [x] 可设置每隔多少分钟(T1)显示一次悬浮窗（默认25，设置0视为关闭功能），每次显示多少秒(T2)后隐藏悬浮窗（默认5）
    - [x] 用户唤出的悬浮窗不应影响周期（T1和T2），也不应反转原本的显示/隐藏行为
## 数据库
- [x] 使用tauri-plugin-sql + SQLite
- [x] 每个任务/子任务是一行记录
- [x] 计时器使用独立的表，每个时间范围占一行，任务id、开始时间、结束时间都需要创建索引以备查询
